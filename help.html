How to Play

Goal

Connect every pair of matching colored dots and fill every cell in the grid to solve the puzzle.

Drawing a path

Press and drag from any colored dot to begin drawing a path. Drag through the grid cells to extend the path toward the matching dot of the same color. Release to pause the path at any point.

Completing a flow

A flow is complete when its path connects both dots of the same color. All flows must be complete and every cell must be filled to win.

Editing a path

Tap any dot to reset that color and start its path over. Tap any cell that already has a color to resume drawing from that point, trimming everything after it. Drag backward along your path to erase the last steps.

Rules

Paths cannot cross each other. Paths cannot loop back on themselves. Paths cannot pass through a dot belonging to a different color.

Difficulty

Easy is a 5×5 grid with 4 colors. Medium is a 7×7 grid with 6 colors. Hard is a 9×9 grid with 8 colors. Use the buttons at the bottom of the game screen to switch difficulty at any time.

Solution

Stuck? Press the Solution button to reveal the answer. Viewing the solution will not count as a win. Press any difficulty button to start a new puzzle.
